Chapter: Arithmetic Progressions | Exercise: 5.3 | Class: 10 | Session: 2026-27

Total Questions: 20 | Topic: Sum of First n Terms of an AP

Class 10 Maths Chapter 5 Exercise 5.3 Solutions 2026-27 English Medium

Key Formulas:

  • Sₙ = n/2 [2a + (n–1)d]
  • Sₙ = n/2 (a + l) where l = last term
  • aₙ = Sₙ – Sₙ₋₁

Question 1

Find the sum of the following APs:

(i) 2, 7, 12, ..., to 10 terms

Solution:

a = 2, d = 5, n = 10

S₁₀ = 10/2 [2(2) + (10–1)(5)] = 5[4 + 45] = 5 × 49 = 245

Answer: S₁₀ = 245

(ii) –37, –33, –29, ..., to 12 terms

Solution:

a = –37, d = 4, n = 12

S₁₂ = 12/2 [2(–37) + (12–1)(4)] = 6[–74 + 44] = 6 × (–30) = –180

Answer: S₁₂ = –180

(iii) 0.6, 1.7, 2.8, ..., to 100 terms

Solution:

a = 0.6, d = 1.1, n = 100

S₁₀₀ = 100/2 [2(0.6) + (99)(1.1)] = 50[1.2 + 108.9] = 50 × 110.1 = 5505

Answer: S₁₀₀ = 5505

(iv) 1/15, 1/12, 1/10, ..., to 11 terms

Solution:

a = 1/15, d = 1/12 – 1/15 = 5/60 – 4/60 = 1/60, n = 11

S₁₁ = 11/2 [2(1/15) + (10)(1/60)]

= 11/2 [2/15 + 10/60] = 11/2 [8/60 + 10/60] = 11/2 × 18/60 = 11/2 × 3/10 = 33/20

Answer: S₁₁ = 33/20

Question 2

Find the sums given below:

(i) 7 + 10½ + 14 + ... + 84

Solution:

a = 7, d = 3.5, l = 84

n = (84–7)/3.5 + 1 = 77/3.5 + 1 = 22 + 1 = 23

Sₙ = n/2(a + l) = 23/2(7 + 84) = 23/2 × 91 = 2093/2 = 1046.5

Answer: Sum = 1046.5

(ii) 34 + 32 + 30 + ... + 10

Solution:

a = 34, d = –2, l = 10

n = (10–34)/(–2) + 1 = 12 + 1 = 13

Sₙ = 13/2(34 + 10) = 13/2 × 44 = 286

Answer: Sum = 286

(iii) –5 + (–8) + (–11) + ... + (–230)

Solution:

a = –5, d = –3, l = –230

n = (–230–(–5))/(–3) + 1 = (–225)/(–3) + 1 = 75 + 1 = 76

Sₙ = 76/2(–5 + (–230)) = 38 × (–235) = –8930

Answer: Sum = –8930

Question 3

In an AP: given a = 5, d = 3, aₙ = 50, find n and Sₙ.

Solution:

aₙ = a + (n–1)d → 50 = 5 + (n–1)3 → 45 = 3(n–1) → n = 16

Sₙ = n/2(a + aₙ) = 16/2(5 + 50) = 8 × 55 = 440

Answer: n = 16, Sₙ = 440

Question 4

In an AP: given a = 7, a₁₃ = 35, find d and S₁₃.

Solution:

a₁₃ = 7 + 12d = 35 → 12d = 28 → d = 7/3

S₁₃ = 13/2(7 + 35) = 13/2 × 42 = 273

Answer: d = 7/3, S₁₃ = 273

Question 5

In an AP: given d = 5, S₉ = 75, find a and a₉.

Solution:

S₉ = 9/2[2a + 8(5)] = 75 → 9/2[2a + 40] = 75

9(2a + 40) = 150 → 2a + 40 = 150/9 → 2a = 150/9 – 40 = –210/9 → a = –35/3

a₉ = –35/3 + 8(5) = –35/3 + 40 = 85/3

Answer: a = –35/3, a₉ = 85/3

Question 6

In an AP: given a = 2, d = 8, Sₙ = 90, find n and aₙ.

Solution:

90 = n/2[2(2) + (n–1)8] = n/2[4 + 8n – 8] = n/2[8n – 4] = n(4n – 2)

4n² – 2n – 90 = 0 → 2n² – n – 45 = 0

(2n + 9)(n – 5) = 0 → n = 5 (taking positive value)

aₙ = 2 + 4(8) = 34

Answer: n = 5, aₙ = 34

Question 7

In an AP: given l = 28, S = 144, and there are total 9 terms. Find a.

Solution:

Sₙ = n/2(a + l) → 144 = 9/2(a + 28)

288 = 9(a + 28) → a + 28 = 32 → a = 4

Answer: a = 4

Question 8

How many terms of the AP: 9, 17, 25, ... must be taken to give a sum of 636?

Solution:

a = 9, d = 8, Sₙ = 636

636 = n/2[18 + (n–1)8] = n/2[8n + 10] = n(4n + 5)

4n² + 5n – 636 = 0

n = (–5 ± √(25 + 10176))/8 = (–5 ± √10201)/8 = (–5 ± 101)/8

n = 96/8 = 12 (taking positive value)

Answer: 12 terms

Question 9

Find the sum of first 22 terms of an AP in which d = 7 and 22nd term is 149.

Solution:

a₂₂ = a + 21(7) = 149 → a = 149 – 147 = 2

S₂₂ = 22/2(2 + 149) = 11 × 151 = 1661

Answer: S₂₂ = 1661

Question 10

Find the sum of first 51 terms of an AP whose second and third terms are 14 and 18 respectively.

Solution:

d = 18 – 14 = 4, a = 14 – 4 = 10

S₅₁ = 51/2[2(10) + 50(4)] = 51/2[20 + 200] = 51/2 × 220 = 5610

Answer: S₅₁ = 5610

Question 11

If the sum of the first n terms of an AP is 4n – n², what is the first term? What is the sum of first two terms? What is the second term? Similarly find the 3rd, the 10th and the nth terms.

Solution:

Sₙ = 4n – n²

S₁ = 4(1) – 1 = 3 → a₁ = 3

S₂ = 4(2) – 4 = 4 → a₂ = S₂ – S₁ = 4 – 3 = 1

S₃ = 12 – 9 = 3 → a₃ = S₃ – S₂ = 3 – 4 = –1

d = a₂ – a₁ = 1 – 3 = –2

a₁₀ = 3 + 9(–2) = 3 – 18 = –15

aₙ = 3 + (n–1)(–2) = 3 – 2n + 2 = 5 – 2n

Answer: a₁ = 3, S₂ = 4, a₂ = 1, a₃ = –1, a₁₀ = –15, aₙ = 5–2n

Question 12

Find the sum of the first 40 positive integers divisible by 6.

Solution:

AP: 6, 12, 18, ..., a = 6, d = 6, n = 40

S₄₀ = 40/2[2(6) + 39(6)] = 20[12 + 234] = 20 × 246 = 4920

Answer: S₄₀ = 4920

Question 13

Find the sum of the first 15 multiples of 8.

Solution:

AP: 8, 16, 24, ..., a = 8, d = 8, n = 15

S₁₅ = 15/2[2(8) + 14(8)] = 15/2[16 + 112] = 15/2 × 128 = 960

Answer: S₁₅ = 960

Question 14

Find the sum of the odd numbers between 0 and 50.

Solution:

AP: 1, 3, 5, ..., 49 → a = 1, d = 2, l = 49

n = (49–1)/2 + 1 = 25

S₂₅ = 25/2(1 + 49) = 25/2 × 50 = 625

Answer: Sum = 625

Question 15

A contract on construction job specifies a penalty for delay of completion beyond a certain date as follows: Rs 200 for the first day, Rs 250 for the second day, Rs 300 for the third day, etc., the penalty for each succeeding day being Rs 50 more than for the preceding day. How much money the contractor has to pay as penalty, if he has delayed the work by 30 days?

Solution:

a = 200, d = 50, n = 30

S₃₀ = 30/2[2(200) + 29(50)] = 15[400 + 1450] = 15 × 1850 = 27750

Answer: Total penalty = Rs 27750

Question 16

A sum of Rs 700 is to be used to give seven cash prizes to students of a school for their overall academic performance. If each prize is Rs 20 less than its preceding prize, find the value of each of the prizes.

Solution:

Let first prize = a, d = –20, n = 7, S₇ = 700

700 = 7/2[2a + 6(–20)] = 7/2[2a – 120]

200 = 2a – 120 → 2a = 320 → a = 160

Prizes: 160, 140, 120, 100, 80, 60, 40

Answer: Prizes are Rs 160, 140, 120, 100, 80, 60, 40.

Question 17

In a school, students thought of planting trees in and around the school to reduce air pollution. It was decided that the number of trees, that each section of each class will plant, will be the same as the class, in which they are studying, e.g., a section of Class I will plant 1 tree, a section of Class II will plant 2 trees and so on till Class XII. There are three sections of each class. How many trees will be planted by the students?

Solution:

Trees planted by each class = 3 × class number

Total trees = 3(1 + 2 + 3 + ... + 12) = 3 × S₁₂

S₁₂ = 12/2(1 + 12) = 6 × 13 = 78

Total = 3 × 78 = 234

Answer: 234 trees will be planted.

Question 18

A spiral is made up of successive semicircles, with centres alternately at A and B, starting with centre at A, of radii 0.5 cm, 1.0 cm, 1.5 cm, 2.0 cm, ... What is the total length of such a spiral made up of thirteen consecutive semicircles?

Solution:

Length of each semicircle = πr

Lengths: π(0.5), π(1.0), π(1.5), ..., π(6.5)

Sum = π(0.5 + 1.0 + 1.5 + ... + 6.5)

AP: 0.5, 1.0, ..., 6.5 → a = 0.5, d = 0.5, n = 13

S₁₃ = 13/2(0.5 + 6.5) = 13/2 × 7 = 45.5

Total length = π × 45.5 = 143/π ≈ 143 cm

Answer: Total length = 143 cm (approximately)

Question 19

200 logs are stacked in the following manner: 20 logs in the bottom row, 19 in the next row, 18 in the row next to it and so on. In how many rows are the 200 logs placed and how many logs are in the top row?

Solution:

a = 20, d = –1, Sₙ = 200

200 = n/2[40 + (n–1)(–1)] = n/2[41 – n]

400 = n(41 – n) = 41n – n²

n² – 41n + 400 = 0

(n – 16)(n – 25) = 0 → n = 16 or n = 25

For n = 25: a₂₅ = 20 + 24(–1) = –4 (not valid, logs can't be negative)

So n = 16

a₁₆ = 20 + 15(–1) = 5

Answer: 16 rows, top row has 5 logs.

Question 20

In a potato race, a bucket is placed at the starting point, which is 5 m from the first potato, and the other potatoes are placed 3 m apart in a straight line. There are ten potatoes in the line. A competitor starts from the bucket, picks up the nearest potato, runs back with it, drops it in the bucket, runs back to pick up the next potato, runs to the bucket to drop it in, and she continues in the same way until all the potatoes are in the bucket. What is the total distance the competitor has to run?

Solution:

Distance to pick 1st potato = 2 × 5 = 10 m

Distance to pick 2nd potato = 2 × 8 = 16 m

Distance to pick 3rd potato = 2 × 11 = 22 m

AP: 10, 16, 22, ..., a = 10, d = 6, n = 10

S₁₀ = 10/2[2(10) + 9(6)] = 5[20 + 54] = 5 × 74 = 370

Answer: Total distance = 370 m

Key Points – Exercise 5.3

  • Sₙ = n/2[2a + (n–1)d]
  • Sₙ = n/2(a + l) when last term l is known.
  • aₙ = Sₙ – Sₙ₋₁ (to find nth term from sum formula).
  • Always check if n is a positive integer — reject negative values.
  • For word problems, identify a, d and what is asked carefully.

Class 10 Maths Chapter 5 Exercise 5.2 Solutions 2026-27 English Medium

Key Formula: nth term of AP = aₙ = a + (n–1)d

Where a = first term, d = common difference, n = term number.

Question 1

Fill in the blanks in the following table, given that a is the first term, d the common difference and aₙ the nth term of the AP:

(i) a = 7, d = 3, n = 8, aₙ = ?

(ii) a = –18, d = ?, n = 10, aₙ = 0

(iii) a = ?, d = –3, n = 18, aₙ = –5

(iv) a = –18.9, d = 2.5, n = ?, aₙ = 3.6

(v) a = 3.5, d = 0, n = 105, aₙ = ?

Solution (i): a = 7, d = 3, n = 8

aₙ = a + (n–1)d = 7 + (8–1) × 3 = 7 + 21 = 28

Answer: aₙ = 28

Solution (ii): a = –18, n = 10, aₙ = 0

0 = –18 + (10–1)d

18 = 9d

Answer: d = 2

Solution (iii): d = –3, n = 18, aₙ = –5

–5 = a + (18–1)(–3)

–5 = a – 51

a = 46

Answer: a = 46

Solution (iv): a = –18.9, d = 2.5, aₙ = 3.6

3.6 = –18.9 + (n–1) × 2.5

3.6 + 18.9 = (n–1) × 2.5

22.5 = (n–1) × 2.5

n–1 = 9 → n = 10

Answer: n = 10

Solution (v): a = 3.5, d = 0, n = 105

aₙ = 3.5 + (105–1) × 0 = 3.5

Answer: aₙ = 3.5

Question 2

Choose the correct choice in the following and justify:

(i) 30th term of the AP: 10, 7, 4, ... is

(A) 97   (B) 77   (C) –77   (D) –87

Solution:

a = 10, d = 7–10 = –3, n = 30

a₃₀ = 10 + (30–1)(–3) = 10 – 87 = –77

Answer: (C) –77

(ii) 11th term of the AP: –3, –1/2, 2, ... is

(A) 28   (B) 22   (C) –38   (D) –48.5

Solution:

a = –3, d = –1/2 – (–3) = 5/2, n = 11

a₁₁ = –3 + (11–1)(5/2) = –3 + 25 = 22

Answer: (B) 22

Question 3

In the following APs, find the missing terms in the boxes:

(i) 2, __, 26    (ii) __, 13, __, 3    (iii) 5, __, __, 9½    (iv) –4, __, __, __, __, 6    (v) __, 38, __, __, __, –22

Solution (i): 2, __, 26

Here a = 2, a₃ = 26

26 = 2 + 2d → 2d = 24 → d = 12

Missing term = 2 + 12 = 14

Answer: 2, 14, 26

Solution (ii): __, 13, __, 3

a₂ = 13, a₄ = 3

a₄ – a₂ = 2d → 3 – 13 = 2d → d = –5

a₁ = 13 – (–5) = 18, a₃ = 13 + (–5) = 8

Answer: 18, 13, 8, 3

Solution (iii): 5, __, __, 9½

a = 5, a₄ = 9.5

9.5 = 5 + 3d → 3d = 4.5 → d = 1.5

a₂ = 6.5, a₃ = 8

Answer: 5, 6.5, 8, 9.5

Solution (iv): –4, __, __, __, __, 6

a = –4, a₆ = 6

6 = –4 + 5d → 5d = 10 → d = 2

Terms: –4, –2, 0, 2, 4, 6

Answer: –4, –2, 0, 2, 4, 6

Solution (v): __, 38, __, __, __, –22

a₂ = 38, a₆ = –22

a₆ – a₂ = 4d → –22 – 38 = 4d → d = –15

a₁ = 38 + 15 = 53

Terms: 53, 38, 23, 8, –7, –22

Answer: 53, 38, 23, 8, –7, –22

Question 4

Which term of the AP: 3, 8, 13, 18, ... is 78?

Solution:

a = 3, d = 5, aₙ = 78

78 = 3 + (n–1) × 5

75 = (n–1) × 5

n–1 = 15 → n = 16

Answer: 78 is the 16th term.

Question 5

Find the number of terms in each of the following APs:

(i) 7, 13, 19, ..., 205    (ii) 18, 15½, 13, ..., –47

Solution (i): 7, 13, 19, ..., 205

a = 7, d = 6, aₙ = 205

205 = 7 + (n–1) × 6

198 = (n–1) × 6

n–1 = 33 → n = 34

Answer: 34 terms

Solution (ii): 18, 15½, 13, ..., –47

a = 18, d = –5/2, aₙ = –47

–47 = 18 + (n–1)(–5/2)

–65 = (n–1)(–5/2)

n–1 = 26 → n = 27

Answer: 27 terms

Question 6

Check whether –150 is a term of the AP: 11, 8, 5, 2, ...

Solution:

a = 11, d = –3, aₙ = –150

–150 = 11 + (n–1)(–3)

–161 = (n–1)(–3)

n–1 = 161/3 = 53.67

n is not a whole number.

Answer: –150 is NOT a term of this AP.

Question 7

Find the 31st term of an AP whose 11th term is 38 and the 16th term is 73.

Solution:

a₁₁ = a + 10d = 38 ... (1)

a₁₆ = a + 15d = 73 ... (2)

Subtract (1) from (2): 5d = 35 → d = 7

From (1): a = 38 – 70 = –32

a₃₁ = –32 + 30 × 7 = –32 + 210 = 178

Answer: 31st term = 178

Question 8

An AP consists of 50 terms of which 3rd term is 12 and the last term is 106. Find the 29th term.

Solution:

a₃ = a + 2d = 12 ... (1)

a₅₀ = a + 49d = 106 ... (2)

Subtract (1) from (2): 47d = 94 → d = 2

From (1): a = 12 – 4 = 8

a₂₉ = 8 + 28 × 2 = 8 + 56 = 64

Answer: 29th term = 64

Question 9

If the 3rd and the 9th terms of an AP are 4 and –8 respectively, which term of this AP is zero?

Solution:

a₃ = a + 2d = 4 ... (1)

a₉ = a + 8d = –8 ... (2)

Subtract (1) from (2): 6d = –12 → d = –2

From (1): a = 4 + 4 = 8

aₙ = 0: 8 + (n–1)(–2) = 0 → n–1 = 4 → n = 5

Answer: 5th term is zero.

Question 10

The 17th term of an AP exceeds its 10th term by 7. Find the common difference.

Solution:

a₁₇ – a₁₀ = 7

(a + 16d) – (a + 9d) = 7

7d = 7

Answer: d = 1

Question 11

Which term of the AP: 3, 15, 27, 39, ... will be 132 more than its 54th term?

Solution:

a = 3, d = 12

a₅₄ = 3 + 53 × 12 = 3 + 636 = 639

aₙ = 639 + 132 = 771

771 = 3 + (n–1) × 12

768 = (n–1) × 12

n–1 = 64 → n = 65

Answer: 65th term is 132 more than the 54th term.

Question 12

Two APs have the same common difference. The difference between their 100th terms is 100. What is the difference between their 1000th terms?

Solution:

Let first APs have first term a₁ and second AP have first term a₂.

a₁₀₀ of AP1 – a₁₀₀ of AP2 = (a₁ + 99d) – (a₂ + 99d) = a₁ – a₂ = 100

a₁₀₀₀ of AP1 – a₁₀₀₀ of AP2 = (a₁ + 999d) – (a₂ + 999d) = a₁ – a₂ = 100

Answer: The difference between their 1000th terms is also 100.

Question 13

How many three-digit numbers are divisible by 7?

Solution:

First 3-digit number divisible by 7 = 105

Last 3-digit number divisible by 7 = 994

AP: 105, 112, ..., 994 with a = 105, d = 7, aₙ = 994

994 = 105 + (n–1) × 7

889 = (n–1) × 7

n–1 = 127 → n = 128

Answer: 128 three-digit numbers are divisible by 7.

Question 14

How many multiples of 4 lie between 10 and 250?

Solution:

First multiple of 4 after 10 = 12

Last multiple of 4 before 250 = 248

AP: 12, 16, ..., 248 with a = 12, d = 4, aₙ = 248

248 = 12 + (n–1) × 4

236 = (n–1) × 4

n–1 = 59 → n = 60

Answer: 60 multiples of 4 lie between 10 and 250.

Question 15

For what value of n, are the nth terms of two APs: 63, 65, 67, ... and 3, 10, 17, ... equal?

Solution:

AP1: a = 63, d = 2 → aₙ = 63 + (n–1) × 2 = 61 + 2n

AP2: a = 3, d = 7 → aₙ = 3 + (n–1) × 7 = –4 + 7n

Set equal: 61 + 2n = –4 + 7n

65 = 5n → n = 13

Answer: n = 13

Question 16

Determine the AP whose 3rd term is 16 and the 7th term exceeds the 5th term by 12.

Solution:

a₇ – a₅ = 12 → 2d = 12 → d = 6

a₃ = a + 2d = 16 → a + 12 = 16 → a = 4

AP: 4, 10, 16, 22, ...

Answer: AP is 4, 10, 16, 22, ...

Question 17

Find the 20th term from the last term of the AP: 3, 8, 13, ..., 253.

Solution:

Last term l = 253, d = 5

20th term from last = l – (20–1)d = 253 – 19 × 5 = 253 – 95 = 158

Answer: 20th term from last = 158

Question 18

The sum of the 4th and 8th terms of an AP is 24 and the sum of the 6th and 10th terms is 44. Find the first three terms of the AP.

Solution:

a₄ + a₈ = 24 → (a+3d) + (a+7d) = 24 → 2a + 10d = 24 → a + 5d = 12 ... (1)

a₆ + a₁₀ = 44 → (a+5d) + (a+9d) = 44 → 2a + 14d = 44 → a + 7d = 22 ... (2)

Subtract (1) from (2): 2d = 10 → d = 5

From (1): a = 12 – 25 = –13

First three terms: –13, –8, –3

Answer: First three terms are –13, –8, –3.

Question 19

Subba Rao started work in 1995 at an annual salary of Rs 5000 and received an increment of Rs 200 each year. In which year did his income reach Rs 7000?

Solution:

a = 5000, d = 200, aₙ = 7000

7000 = 5000 + (n–1) × 200

2000 = (n–1) × 200

n–1 = 10 → n = 11

Year = 1995 + 10 = 2005

Answer: His income reached Rs 7000 in the year 2005.

Question 20

Ramkali saved Rs 5 in the first week of a year and then increased her weekly savings by Rs 1.75. If in the nth week, her weekly savings become Rs 20.75, find n.

Solution:

a = 5, d = 1.75, aₙ = 20.75

20.75 = 5 + (n–1) × 1.75

15.75 = (n–1) × 1.75

n–1 = 9 → n = 10

Answer: n = 10

Key Points – Exercise 5.2

  • aₙ = a + (n–1)d
  • To find n: rearrange formula and solve.
  • If n comes out as a fraction → that value is NOT a term of the AP.
  • nth term from last = l – (n–1)d where l = last term.
  • If two APs have same d, difference between their nth terms is always constant.

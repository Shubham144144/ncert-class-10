What is an Arithmetic Progression (AP)?

A list of numbers in which each term is obtained by adding a fixed number to the previous term is called an Arithmetic Progression.

The fixed number is called the Common Difference (d).

How to check if a list is an AP:

  • Find the difference between consecutive terms.
  • If all differences are equal → it is an AP.
  • If differences are not equal → it is NOT an AP.

General form of AP: a, a+d, a+2d, a+3d, ...

Where a = first term, d = common difference.

Question 1

Question:

In which of the following situations, does the list of numbers involved make an arithmetic progression, and why?

(i) The taxi fare after each km when the fare is Rs 15 for the first km and Rs 8 for each additional km.

(ii) The amount of air present in a cylinder when a vacuum pump removes 1/4 of the air remaining in the cylinder at a time.

(iii) The cost of digging a well after every metre of digging, when it costs Rs 150 for the first metre and rises by Rs 50 for each subsequent metre.

(iv) The amount of money in the account every year, when Rs 10000 is deposited at compound interest at 8% per annum.

Solution (i): Taxi fare

Step 1: List the fares after each km.

After 1st km = Rs 15

After 2nd km = 15 + 8 = Rs 23

After 3rd km = 23 + 8 = Rs 31

After 4th km = 31 + 8 = Rs 39

List: 15, 23, 31, 39, ...

Step 2: Check differences.

23 – 15 = 8, 31 – 23 = 8, 39 – 31 = 8

All differences are equal to 8.

Answer: Yes, it is an AP. First term a = 15, Common difference d = 8.

Solution (ii): Air in cylinder

Step 1: Let initial air = 1 unit.

After 1st pump = 1 – 1/4 = 3/4

After 2nd pump = 3/4 – 1/4 × 3/4 = 3/4 × 3/4 = 9/16

After 3rd pump = 9/16 × 3/4 = 27/64

List: 1, 3/4, 9/16, 27/64, ...

Step 2: Check differences.

3/4 – 1 = –1/4

9/16 – 3/4 = 9/16 – 12/16 = –3/16

Differences are NOT equal (–1/4 ≠ –3/16).

Answer: No, it is NOT an AP.

Solution (iii): Cost of digging a well

Step 1: List the costs.

After 1st metre = Rs 150

After 2nd metre = 150 + 50 = Rs 200

After 3rd metre = 200 + 50 = Rs 250

After 4th metre = 250 + 50 = Rs 300

List: 150, 200, 250, 300, ...

Step 2: Check differences.

200 – 150 = 50, 250 – 200 = 50, 300 – 250 = 50

All differences are equal to 50.

Answer: Yes, it is an AP. First term a = 150, Common difference d = 50.

Solution (iv): Compound interest

Step 1: Amount after each year at 8% compound interest.

After year 1 = 10000 × (1.08)¹ = Rs 10800

After year 2 = 10000 × (1.08)² = Rs 11664

After year 3 = 10000 × (1.08)³ = Rs 12597.12

Step 2: Check differences.

11664 – 10800 = 864

12597.12 – 11664 = 933.12

Differences are NOT equal.

Answer: No, it is NOT an AP. (Compound interest grows exponentially, not linearly.)

Question 2

Question:

Write first four terms of the AP, when the first term a and the common difference d are given as follows:

(i) a = 10, d = 10    (ii) a = –2, d = 0    (iii) a = 4, d = –3    (iv) a = –1, d = 1/2    (v) a = –1.25, d = –0.25

Solution (i): a = 10, d = 10

1st term = 10

2nd term = 10 + 10 = 20

3rd term = 20 + 10 = 30

4th term = 30 + 10 = 40

Answer: 10, 20, 30, 40

Solution (ii): a = –2, d = 0

1st term = –2

2nd term = –2 + 0 = –2

3rd term = –2 + 0 = –2

4th term = –2 + 0 = –2

Answer: –2, –2, –2, –2 (constant AP)

Solution (iii): a = 4, d = –3

1st term = 4

2nd term = 4 + (–3) = 1

3rd term = 1 + (–3) = –2

4th term = –2 + (–3) = –5

Answer: 4, 1, –2, –5

Solution (iv): a = –1, d = 1/2

1st term = –1

2nd term = –1 + 1/2 = –1/2

3rd term = –1/2 + 1/2 = 0

4th term = 0 + 1/2 = 1/2

Answer: –1, –1/2, 0, 1/2

Solution (v): a = –1.25, d = –0.25

1st term = –1.25

2nd term = –1.25 + (–0.25) = –1.50

3rd term = –1.50 + (–0.25) = –1.75

4th term = –1.75 + (–0.25) = –2.00

Answer: –1.25, –1.50, –1.75, –2.00

Question 3

Question:

For the following APs, write the first term and the common difference:

(i) 3, 1, –1, –3, ...    (ii) –5, –1, 3, 7, ...    (iii) 1/3, 5/3, 9/3, 13/3, ...    (iv) 0.6, 1.7, 2.8, 3.9, ...

Solution (i): 3, 1, –1, –3, ...

First term a = 3

Common difference d = 1 – 3 = –2

Answer: a = 3, d = –2

Solution (ii): –5, –1, 3, 7, ...

First term a = –5

Common difference d = –1 – (–5) = 4

Answer: a = –5, d = 4

Solution (iii): 1/3, 5/3, 9/3, 13/3, ...

First term a = 1/3

Common difference d = 5/3 – 1/3 = 4/3

Answer: a = 1/3, d = 4/3

Solution (iv): 0.6, 1.7, 2.8, 3.9, ...

First term a = 0.6

Common difference d = 1.7 – 0.6 = 1.1

Answer: a = 0.6, d = 1.1

Question 4

Question:

Which of the following are APs? If they form an AP, find the common difference d and write three more terms.

(i) 2, 4, 8, 16, ...

(ii) 2, 5/2, 3, 7/2, ...

(iii) –1.2, –3.2, –5.2, –7.2, ...

(iv) –10, –6, –2, 2, ...

(v) 3, 3+√2, 3+2√2, 3+3√2, ...

(vi) 0.2, 0.22, 0.222, 0.2222, ...

(vii) 0, –4, –8, –12, ...

(viii) –1/2, –1/2, –1/2, –1/2, ...

(ix) 1, 3, 9, 27, ...

(x) a, 2a, 3a, 4a, ...

(xi) a, a², a³, a⁴, ...

(xii) √2, √8, √18, √32, ...

(xiii) √3, √6, √9, √12, ...

(xiv) 1², 3², 5², 7², ...

(xv) 1², 5², 7², 73, ...

Solution (i): 2, 4, 8, 16, ...

4–2 = 2, 8–4 = 4. Differences NOT equal.

Not an AP.

Solution (ii): 2, 5/2, 3, 7/2, ...

5/2 – 2 = 1/2, 3 – 5/2 = 1/2, 7/2 – 3 = 1/2. Equal!

d = 1/2. Next three terms: 4, 9/2, 5

Yes, AP. d = 1/2. Next terms: 4, 9/2, 5

Solution (iii): –1.2, –3.2, –5.2, –7.2, ...

–3.2–(–1.2) = –2, –5.2–(–3.2) = –2. Equal!

d = –2. Next three terms: –9.2, –11.2, –13.2

Yes, AP. d = –2. Next terms: –9.2, –11.2, –13.2

Solution (iv): –10, –6, –2, 2, ...

–6–(–10) = 4, –2–(–6) = 4, 2–(–2) = 4. Equal!

d = 4. Next three terms: 6, 10, 14

Yes, AP. d = 4. Next terms: 6, 10, 14

Solution (v): 3, 3+√2, 3+2√2, 3+3√2, ...

(3+√2)–3 = √2, (3+2√2)–(3+√2) = √2. Equal!

d = √2. Next three terms: 3+4√2, 3+5√2, 3+6√2

Yes, AP. d = √2. Next terms: 3+4√2, 3+5√2, 3+6√2

Solution (vi): 0.2, 0.22, 0.222, 0.2222, ...

0.22–0.2 = 0.02, 0.222–0.22 = 0.002. NOT equal.

Not an AP.

Solution (vii): 0, –4, –8, –12, ...

–4–0 = –4, –8–(–4) = –4. Equal!

d = –4. Next three terms: –16, –20, –24

Yes, AP. d = –4. Next terms: –16, –20, –24

Solution (viii): –1/2, –1/2, –1/2, –1/2, ...

–1/2–(–1/2) = 0. Equal!

d = 0. Next three terms: –1/2, –1/2, –1/2

Yes, AP (constant). d = 0. Next terms: –1/2, –1/2, –1/2

Solution (ix): 1, 3, 9, 27, ...

3–1 = 2, 9–3 = 6. NOT equal.

Not an AP.

Solution (x): a, 2a, 3a, 4a, ...

2a–a = a, 3a–2a = a. Equal!

d = a. Next three terms: 5a, 6a, 7a

Yes, AP. d = a. Next terms: 5a, 6a, 7a

Solution (xi): a, a², a³, a⁴, ...

a²–a = a(a–1), a³–a² = a²(a–1). Equal only if a = 0 or a = 1.

Not an AP (in general).

Solution (xii): √2, √8, √18, √32, ...

√2, 2√2, 3√2, 4√2, ...

2√2–√2 = √2, 3√2–2√2 = √2. Equal!

d = √2. Next three terms: 5√2, 6√2, 7√2 = √50, √72, √98

Yes, AP. d = √2. Next terms: √50, √72, √98

Solution (xiii): √3, √6, √9, √12, ...

√6–√3 = √3(√2–1), √9–√6 = √3(√3–√2). NOT equal.

Not an AP.

Solution (xiv): 1², 3², 5², 7², ... = 1, 9, 25, 49, ...

9–1 = 8, 25–9 = 16. NOT equal.

Not an AP.

Solution (xv): 1², 5², 7², 73 = 1, 25, 49, 73, ...

25–1 = 24, 49–25 = 24, 73–49 = 24. Equal!

d = 24. Next three terms: 97, 121, 145

Yes, AP. d = 24. Next terms: 97, 121, 145

Key Points – Exercise 5.1

  • Check if differences between consecutive terms are equal.
  • d = a₂ – a₁ = a₃ – a₂ = a₄ – a₃
  • d can be positive, negative, or zero.
  • Compound interest and geometric sequences are NOT APs.
  • Simple interest and linear growth ARE APs.

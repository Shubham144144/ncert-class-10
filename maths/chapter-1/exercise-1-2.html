Exercise 1.2 — Real Numbers

NCERT Class 10 Maths, Chapter 1

Key Concepts

These proofs use the "proof by contradiction" method along with the theorem: if a prime p divides a2, then p also divides a. This exercise builds on the irrationality of √2, proved earlier in the chapter.

Solutions

Q1. Prove that √5 is irrational.
Why this answer: assume the opposite (that √5 is rational) and show it leads to a contradiction.
Let us assume, to the contrary, that √5 is rational. Then we can write √5 = pq for some coprime integers p and q (q ≠ 0), meaning p and q have no common factor other than 1.

So, √5 q = p. Squaring both sides: 5q2 = p2.
This means 5 divides p2, and therefore 5 also divides p.

So we can write p = 5m for some integer m. Substituting back: 5q2 = (5m)2 = 25m2, so q2 = 5m2.
This means 5 divides q2, and therefore 5 also divides q.

So 5 divides both p and q, which contradicts our assumption that p and q are coprime.
Answer: This contradiction shows that √5 is irrational.
Common mistake: forgetting to explicitly state "p and q are coprime" (or "have no common factor other than 1") at the start — CBSE marking checks for this assumption, since the whole proof depends on contradicting it.
Q2. Prove that 3 + 2√5 is irrational.
Why this answer: if 3 + 2√5 were rational, rearranging it would force √5 to be rational too — but Q1 already proved that's false.
Let us assume, to the contrary, that 3 + 2√5 is rational. Then we can write 3 + 2√5 = ab for some coprime integers a and b (b ≠ 0).

Rearranging: 2√5 = ab − 3 = a − 3bb
So, √5 = a − 3b2b
Since a and b are integers, the right-hand side is rational. So √5 must be rational.
But this contradicts the fact (proved in Q1) that √5 is irrational.
Answer: This contradiction shows that 3 + 2√5 is irrational.
Q3. Prove that the following are irrationals:
Why this answer: the same rearrangement trick as Q2 — isolate √2 and show it would have to be rational.
(i) 1√2
Let us assume, to the contrary, that 1√2 is rational. Then 1√2 = pq for coprime integers p, q (q ≠ 0).
Rearranging: √2 = qp, which is rational since p and q are integers.
This contradicts the known fact that √2 is irrational.
Answer: Contradiction — so 1/√2 is irrational.
(ii) 7√5
Let us assume, to the contrary, that 7√5 is rational. Then 7√5 = pq for coprime integers p, q (q ≠ 0).
Rearranging: √5 = p7q, which is rational.
This contradicts Q1, which proved √5 is irrational.
Answer: Contradiction — so 7√5 is irrational.
(iii) 6 + √2
Let us assume, to the contrary, that 6 + √2 is rational. Then 6 + √2 = pq for coprime integers p, q (q ≠ 0).
Rearranging: √2 = pq − 6 = p − 6qq, which is rational.
This contradicts the known fact that √2 is irrational.
Answer: Contradiction — so 6 + √2 is irrational.
Common mistake: writing "√2 is irrational, so adding/multiplying by a rational number gives an irrational" as if it's an obvious rule — CBSE wants the full contradiction proof spelled out each time, not just this rule stated.

Key Points

  • To prove a number is irrational, assume the opposite (that it's rational, i.e. expressible as p/q with p, q coprime) and derive a contradiction.
  • If a prime p divides a2, then p divides a — this step is essential in most of these proofs.
  • Once √2, √3, √5 etc. are known to be irrational, any expression built from them (like 3 + 2√5 or 7√5) can be proved irrational by isolating the root and showing it would otherwise have to be rational.

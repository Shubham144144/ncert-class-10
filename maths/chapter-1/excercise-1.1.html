Topic Covered: Euclid's Division Lemma and Euclid's Division Algorithm

Euclid's Division Lemma (Simple Meaning): For any two positive integers a and b, we can always find two whole numbers q (quotient) and r (remainder) such that:

a = bq + r, where 0 ≤ r < b

Question 1

Question:

Use Euclid's division algorithm to find the HCF of:

(i) 135 and 225

(ii) 196 and 38220

(iii) 867 and 255

Solution (i): HCF of 135 and 225

Step 1: Here, 225 > 135. So we apply Euclid's division lemma to 225 and 135.

Divide 225 by 135:

225 = 135 × 1 + 90

Remainder = 90 (not zero, so continue)

Step 2: Now apply the lemma to 135 and 90.

Divide 135 by 90:

135 = 90 × 1 + 45

Remainder = 45 (not zero, so continue)

Step 3: Now apply the lemma to 90 and 45.

Divide 90 by 45:

90 = 45 × 2 + 0

Remainder = 0. Stop here.

Answer: The last divisor when remainder became 0 is 45.

Therefore, HCF (135, 225) = 45

Solution (ii): HCF of 196 and 38220

Step 1: Here, 38220 > 196. So we apply Euclid's division lemma to 38220 and 196.

Divide 38220 by 196:

38220 = 196 × 195 + 0

Remainder = 0. Stop here.

Step 2: Since the remainder is 0 in the very first step, the divisor itself is the HCF.

Answer: HCF (196, 38220) = 196

Solution (iii): HCF of 867 and 255

Step 1: Here, 867 > 255. Apply Euclid's division lemma to 867 and 255.

Divide 867 by 255:

867 = 255 × 3 + 102

Remainder = 102 (not zero, so continue)

Step 2: Now apply the lemma to 255 and 102.

Divide 255 by 102:

255 = 102 × 2 + 51

Remainder = 51 (not zero, so continue)

Step 3: Now apply the lemma to 102 and 51.

Divide 102 by 51:

102 = 51 × 2 + 0

Remainder = 0. Stop here.

Answer: HCF (867, 255) = 51

Question 2

Question:

Show that any positive odd integer is of the form 6q + 1, or 6q + 3, or 6q + 5, where q is some integer.

Solution:

Step 1: Let a be any positive integer. Let b = 6.

By Euclid's division lemma: a = 6q + r, where 0 ≤ r < 6.

So the possible remainders are: r = 0, 1, 2, 3, 4, or 5.

Step 2: This means a can be one of these:

6q, 6q + 1, 6q + 2, 6q + 3, 6q + 4, or 6q + 5

Step 3: Now find which of these are odd numbers.

6q = 2(3q) → Even

6q + 1 = 2(3q) + 1 → Odd

6q + 2 = 2(3q + 1) → Even

6q + 3 = 2(3q + 1) + 1 → Odd

6q + 4 = 2(3q + 2) → Even

6q + 5 = 2(3q + 2) + 1 → Odd

Step 4: So the odd forms are only: 6q + 1, 6q + 3, and 6q + 5.

Answer: Any positive odd integer must be of the form 6q + 1, 6q + 3, or 6q + 5. Hence proved.

Question 3

Question:

An army contingent of 616 members is to march behind an army band of 32 members in a parade. The two groups are to march in the same number of columns. What is the maximum number of columns in which they can march?

Solution:

Step 1: To find the maximum number of columns, we need to find the HCF of 616 and 32.

Step 2: Apply Euclid's division algorithm. Here 616 > 32.

Divide 616 by 32:

616 = 32 × 19 + 8

Remainder = 8 (not zero, continue)

Step 3: Now divide 32 by 8:

32 = 8 × 4 + 0

Remainder = 0. Stop here.

Answer: HCF (616, 32) = 8

The maximum number of columns in which they can march is 8.

Question 4

Question:

Use Euclid's division lemma to show that the square of any positive integer is either of the form 3m or 3m + 1 for some integer m.

[Hint: Let x be any positive integer then it is of the form 3q, 3q + 1 or 3q + 2. Now square each of these and show that they can be rewritten in the form 3m or 3m + 1.]

Solution:

Step 1: Let x be any positive integer and b = 3.

By Euclid's lemma: x = 3q + r, where r = 0, 1, or 2.

So x can be: 3q, 3q + 1, or 3q + 2.

Step 2: Case 1 – When x = 3q

x² = (3q)² = 9q² = 3(3q²)

Let m = 3q², then x² = 3m

This is of the form 3m.

Step 3: Case 2 – When x = 3q + 1

x² = (3q + 1)²

= 9q² + 6q + 1

= 3(3q² + 2q) + 1

Let m = 3q² + 2q, then x² = 3m + 1

This is of the form 3m + 1.

Step 4: Case 3 – When x = 3q + 2

x² = (3q + 2)²

= 9q² + 12q + 4

= 9q² + 12q + 3 + 1

= 3(3q² + 4q + 1) + 1

Let m = 3q² + 4q + 1, then x² = 3m + 1

This is of the form 3m + 1.

Answer: In all three cases, x² is either of the form 3m or 3m + 1. Hence proved.

Question 5

Question:

Use Euclid's division lemma to show that the cube of any positive integer is of the form 9m, 9m + 1 or 9m + 8.

Solution:

Step 1: Let a be any positive integer and b = 3.

By Euclid's lemma: a = 3q + r, where r = 0, 1, or 2.

So a can be: 3q, 3q + 1, or 3q + 2.

Step 2: Case 1 – When a = 3q

a³ = (3q)³ = 27q³ = 9(3q³)

Let m = 3q³, then a³ = 9m

This is of the form 9m.

Step 3: Case 2 – When a = 3q + 1

a³ = (3q + 1)³

= 27q³ + 27q² + 9q + 1

= 9(3q³ + 3q² + q) + 1

Let m = 3q³ + 3q² + q, then a³ = 9m + 1

This is of the form 9m + 1.

Step 4: Case 3 – When a = 3q + 2

a³ = (3q + 2)³

= 27q³ + 54q² + 36q + 8

= 9(3q³ + 6q² + 4q) + 8

Let m = 3q³ + 6q² + 4q, then a³ = 9m + 8

This is of the form 9m + 8.

Answer: In all three cases, the cube of any positive integer is of the form 9m, 9m + 1, or 9m + 8. Hence proved.

Key Points to Remember

  • Euclid's Division Lemma: For any two positive integers a and b, a = bq + r, where 0 ≤ r < b.
  • HCF by Euclid's Algorithm: Keep applying the lemma until remainder = 0. The last non-zero remainder is the HCF.
  • Always divide the larger number by the smaller number first.
  • If the remainder is 0 in the first step itself, then the smaller number is the HCF.

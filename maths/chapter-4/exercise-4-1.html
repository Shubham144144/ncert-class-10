What is a Quadratic Equation?

An equation of the form ax² + bx + c = 0, where a, b, c are real numbers and a ≠ 0, is called a quadratic equation.

Key Points:

  • The highest power of the variable must be 2.
  • The coefficient of x² (that is, a) must NOT be zero.
  • Example: 2x² – 3x + 1 = 0 is a quadratic equation.
  • Example: x³ + 2x = 0 is NOT a quadratic equation (degree is 3).
  • Example: x + 5 = 0 is NOT a quadratic equation (degree is 1).

Question 1

Question:

Check whether the following are quadratic equations:

(i) (x + 1)² = 2(x – 3)

(ii) x² – 2x = (–2)(3 – x)

(iii) (x – 2)(x + 1) = (x – 1)(x + 3)

(iv) (x – 3)(2x + 1) = x(x + 5)

(v) (2x – 1)(x – 3) = (x + 5)(x – 1)

(vi) x² + 3x + 1 = (x – 2)²

(vii) (x + 2)³ = 2x(x² – 1)

(viii) x³ – 4x² – x + 1 = (x – 2)³

Solution (i): (x + 1)² = 2(x – 3)

Step 1: Expand both sides.

x² + 2x + 1 = 2x – 6

Step 2: Bring all terms to left side.

x² + 2x + 1 – 2x + 6 = 0

x² + 7 = 0

Step 3: This is of the form ax² + bx + c = 0 where a = 1, b = 0, c = 7.

Answer: Yes, it is a quadratic equation.

Solution (ii): x² – 2x = (–2)(3 – x)

Step 1: Expand right side.

x² – 2x = –6 + 2x

Step 2: Bring all terms to left side.

x² – 2x – 2x + 6 = 0

x² – 4x + 6 = 0

Step 3: This is of the form ax² + bx + c = 0 where a = 1, b = –4, c = 6.

Answer: Yes, it is a quadratic equation.

Solution (iii): (x – 2)(x + 1) = (x – 1)(x + 3)

Step 1: Expand both sides.

x² + x – 2x – 2 = x² + 3x – x – 3

x² – x – 2 = x² + 2x – 3

Step 2: Bring all terms to left side.

x² – x – 2 – x² – 2x + 3 = 0

–3x + 1 = 0

Step 3: The x² terms cancelled. This is a linear equation, not quadratic.

Answer: No, it is NOT a quadratic equation.

Solution (iv): (x – 3)(2x + 1) = x(x + 5)

Step 1: Expand both sides.

2x² + x – 6x – 3 = x² + 5x

2x² – 5x – 3 = x² + 5x

Step 2: Bring all terms to left side.

2x² – 5x – 3 – x² – 5x = 0

x² – 10x – 3 = 0

Step 3: This is of the form ax² + bx + c = 0 where a = 1, b = –10, c = –3.

Answer: Yes, it is a quadratic equation.

Solution (v): (2x – 1)(x – 3) = (x + 5)(x – 1)

Step 1: Expand both sides.

2x² – 6x – x + 3 = x² – x + 5x – 5

2x² – 7x + 3 = x² + 4x – 5

Step 2: Bring all terms to left side.

2x² – 7x + 3 – x² – 4x + 5 = 0

x² – 11x + 8 = 0

Step 3: This is of the form ax² + bx + c = 0 where a = 1, b = –11, c = 8.

Answer: Yes, it is a quadratic equation.

Solution (vi): x² + 3x + 1 = (x – 2)²

Step 1: Expand right side.

x² + 3x + 1 = x² – 4x + 4

Step 2: Bring all terms to left side.

x² + 3x + 1 – x² + 4x – 4 = 0

7x – 3 = 0

Step 3: The x² terms cancelled. This is a linear equation.

Answer: No, it is NOT a quadratic equation.

Solution (vii): (x + 2)³ = 2x(x² – 1)

Step 1: Expand both sides.

x³ + 6x² + 12x + 8 = 2x³ – 2x

Step 2: Bring all terms to left side.

x³ + 6x² + 12x + 8 – 2x³ + 2x = 0

–x³ + 6x² + 14x + 8 = 0

Step 3: The highest degree is 3. This is a cubic equation.

Answer: No, it is NOT a quadratic equation.

Solution (viii): x³ – 4x² – x + 1 = (x – 2)³

Step 1: Expand right side.

(x – 2)³ = x³ – 6x² + 12x – 8

Step 2: Bring all terms to left side.

x³ – 4x² – x + 1 – x³ + 6x² – 12x + 8 = 0

2x² – 13x + 9 = 0

Step 3: This is of the form ax² + bx + c = 0 where a = 2, b = –13, c = 9.

Answer: Yes, it is a quadratic equation.

Summary of Question 1

(i) ✅ Yes — x² + 7 = 0

(ii) ✅ Yes — x² – 4x + 6 = 0

(iii) ❌ No — linear equation

(iv) ✅ Yes — x² – 10x – 3 = 0

(v) ✅ Yes — x² – 11x + 8 = 0

(vi) ❌ No — linear equation

(vii) ❌ No — cubic equation

(viii) ✅ Yes — 2x² – 13x + 9 = 0

Question 2

Question:

Represent the following situations in the form of quadratic equations:

(i) The area of a rectangular plot is 528 m². The length of the plot (in metres) is one more than twice its breadth. We need to find the length and breadth of the plot.

(ii) The product of two consecutive positive integers is 306. We need to find the integers.

(iii) Rohan's mother is 26 years older than him. The product of their ages (in years) 3 years from now will be 360. We would like to find Rohan's present age.

(iv) A train travels a distance of 480 km at a uniform speed. If the speed had been 8 km/h less, then it would have taken 3 hours more to cover the same distance. We need to find the speed of the train.

Solution (i): Rectangular plot area = 528 m²

Step 1: Let breadth = x metres.

Then length = 2x + 1 metres.

Step 2: Area = length × breadth

528 = (2x + 1) × x

528 = 2x² + x

Answer: 2x² + x – 528 = 0

Solution (ii): Product of two consecutive positive integers = 306

Step 1: Let first integer = x.

Then second integer = x + 1.

Step 2: Product = x(x + 1) = 306

x² + x = 306

Answer: x² + x – 306 = 0

Solution (iii): Rohan's mother is 26 years older

Step 1: Let Rohan's present age = x years.

Mother's present age = x + 26 years.

Step 2: 3 years from now:

Rohan's age = x + 3, Mother's age = x + 29

Product = (x + 3)(x + 29) = 360

x² + 29x + 3x + 87 = 360

x² + 32x + 87 – 360 = 0

Answer: x² + 32x – 273 = 0

Solution (iv): Train travels 480 km

Step 1: Let speed of train = x km/h.

Time taken = 480/x hours.

Step 2: If speed = x – 8, time = 480/(x – 8)

New time – Original time = 3 hours

480/(x – 8) – 480/x = 3

Step 3: Multiply throughout by x(x – 8):

480x – 480(x – 8) = 3x(x – 8)

480x – 480x + 3840 = 3x² – 24x

3840 = 3x² – 24x

3x² – 24x – 3840 = 0

Divide by 3:

Answer: x² – 8x – 1280 = 0

Key Points – Exercise 4.1

  • Standard form: ax² + bx + c = 0, a ≠ 0.
  • Always expand and simplify before checking.
  • If x² terms cancel — it is NOT a quadratic equation.
  • If degree is more than 2 — it is NOT a quadratic equation.
  • For word problems — assign a variable, form the equation, simplify.

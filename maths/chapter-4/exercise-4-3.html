Class 10 Maths Chapter 4 Exercise 4.3 Solutions 2026-27 English Medium

Key Formulas:

  • Quadratic Formula: x = (–b ± √(b²–4ac)) / 2a
  • Discriminant: D = b² – 4ac
  • D > 0 → Two distinct real roots
  • D = 0 → Two equal real roots (x = –b/2a)
  • D < 0 → No real roots

Question 1

Question:

Find the nature of the roots of the following quadratic equations. If the real roots exist, find them:

(i) 2x² – 3x + 5 = 0

(ii) 3x² – 4√3 x + 4 = 0

(iii) 2x² – 6x + 3 = 0

Solution (i): 2x² – 3x + 5 = 0

Step 1: Here a = 2, b = –3, c = 5

Step 2: Find discriminant D = b² – 4ac

D = (–3)² – 4(2)(5)

D = 9 – 40 = –31

Step 3: Since D = –31 < 0

Answer: No real roots exist.

Solution (ii): 3x² – 4√3 x + 4 = 0

Step 1: Here a = 3, b = –4√3, c = 4

Step 2: D = b² – 4ac

D = (–4√3)² – 4(3)(4)

D = 48 – 48 = 0

Step 3: Since D = 0 → Two equal real roots.

Step 4: x = –b/2a = 4√3/6 = 2√3/3 = 2/√3

Answer: Two equal roots. x = 2/√3 = 2√3/3

Solution (iii): 2x² – 6x + 3 = 0

Step 1: Here a = 2, b = –6, c = 3

Step 2: D = b² – 4ac

D = (–6)² – 4(2)(3)

D = 36 – 24 = 12

Step 3: Since D = 12 > 0 → Two distinct real roots.

Step 4: x = (–b ± √D) / 2a

x = (6 ± √12) / 4

x = (6 ± 2√3) / 4

x = (3 ± √3) / 2

x = (3 + √3)/2 or x = (3 – √3)/2

Answer: Two distinct real roots. x = (3+√3)/2 and x = (3–√3)/2

Question 2

Question:

Find the values of k for each of the following quadratic equations, so that they have two equal roots:

(i) 2x² + kx + 3 = 0

(ii) kx(x – 2) + 6 = 0

Solution (i): 2x² + kx + 3 = 0

Step 1: For equal roots, D = 0.

Here a = 2, b = k, c = 3

Step 2: D = b² – 4ac = 0

k² – 4(2)(3) = 0

k² – 24 = 0

k² = 24

k = ±√24 = ±2√6

Answer: k = 2√6 or k = –2√6

Solution (ii): kx(x – 2) + 6 = 0

Step 1: Expand.

kx² – 2kx + 6 = 0

Here a = k, b = –2k, c = 6

Step 2: For equal roots, D = 0.

(–2k)² – 4(k)(6) = 0

4k² – 24k = 0

4k(k – 6) = 0

k = 0 or k = 6

But k = 0 makes the equation non-quadratic. So k ≠ 0.

Answer: k = 6

Question 3

Question:

Is it possible to design a rectangular mango grove whose length is twice its breadth, and the area is 800 m²? If so, find its length and breadth.

Solution:

Step 1: Let breadth = x metres. Then length = 2x metres.

Step 2: Area = length × breadth = 800

2x × x = 800

2x² = 800

x² = 400

x = 20 (taking positive value)

Step 3: Check discriminant.

2x² – 800 = 0 → a = 2, b = 0, c = –800

D = 0 – 4(2)(–800) = 6400 > 0 → Real roots exist.

Answer: Yes, it is possible. Breadth = 20 m, Length = 40 m.

Question 4

Question:

Is the following situation possible? If so, determine their present ages. The sum of the ages of two friends is 20 years. Four years ago, the product of their ages in years was 48.

Solution:

Step 1: Let age of first friend = x years. Second = 20 – x years.

Step 2: Four years ago:

First = x – 4, Second = 20 – x – 4 = 16 – x

Product = 48

(x – 4)(16 – x) = 48

16x – x² – 64 + 4x = 48

–x² + 20x – 64 = 48

–x² + 20x – 112 = 0

x² – 20x + 112 = 0

Step 3: Check discriminant.

a = 1, b = –20, c = 112

D = (–20)² – 4(1)(112) = 400 – 448 = –48

D < 0 → No real roots.

Answer: This situation is NOT possible.

Question 5

Question:

Is it possible to design a rectangular park of perimeter 80 m and area 400 m²? If so, find its length and breadth.

Solution:

Step 1: Let length = x metres, breadth = y metres.

Perimeter: 2(x + y) = 80 → x + y = 40 → y = 40 – x

Step 2: Area = x × y = 400

x(40 – x) = 400

40x – x² = 400

x² – 40x + 400 = 0

Step 3: Check discriminant.

a = 1, b = –40, c = 400

D = (–40)² – 4(1)(400) = 1600 – 1600 = 0

D = 0 → Two equal real roots. Design is possible!

Step 4: x = –b/2a = 40/2 = 20

y = 40 – 20 = 20

Answer: Yes, it is possible. Length = 20 m, Breadth = 20 m. (It is a square park.)

Key Points – Exercise 4.3

  • Always find D = b² – 4ac first.
  • D > 0 → 2 distinct real roots → use quadratic formula.
  • D = 0 → 2 equal roots → x = –b/2a.
  • D < 0 → No real roots → situation not possible.
  • For equal roots condition → set D = 0 and solve for k.
  • Quadratic formula: x = (–b ± √D) / 2a

Factorisation Method — How it works:

  • Step 1: Write equation in standard form ax² + bx + c = 0.
  • Step 2: Find two numbers whose sum = b and product = ac.
  • Step 3: Split the middle term using these two numbers.
  • Step 4: Factorise by grouping.
  • Step 5: Set each factor = 0 and solve.

Question 1

Question:

Find the roots of the following quadratic equations by factorisation:

(i) x² – 3x – 10 = 0

(ii) 2x² + x – 6 = 0

(iii) √2 x² + 7x + 5√2 = 0

(iv) 2x² – x + 1/8 = 0

(v) 100x² – 20x + 1 = 0

Solution (i): x² – 3x – 10 = 0

Step 1: Find two numbers with sum = –3 and product = –10.

Numbers are –5 and 2. (–5 + 2 = –3, –5 × 2 = –10)

Step 2: Split middle term.

x² – 5x + 2x – 10 = 0

Step 3: Factorise by grouping.

x(x – 5) + 2(x – 5) = 0

(x + 2)(x – 5) = 0

Step 4: Set each factor = 0.

x + 2 = 0 → x = –2

x – 5 = 0 → x = 5

Answer: Roots are x = –2 and x = 5.

Solution (ii): 2x² + x – 6 = 0

Step 1: Find two numbers with sum = 1 and product = 2 × (–6) = –12.

Numbers are 4 and –3. (4 + (–3) = 1, 4 × (–3) = –12)

Step 2: Split middle term.

2x² + 4x – 3x – 6 = 0

Step 3: Factorise by grouping.

2x(x + 2) – 3(x + 2) = 0

(2x – 3)(x + 2) = 0

Step 4: Set each factor = 0.

2x – 3 = 0 → x = 3/2

x + 2 = 0 → x = –2

Answer: Roots are x = 3/2 and x = –2.

Solution (iii): √2 x² + 7x + 5√2 = 0

Step 1: Find two numbers with sum = 7 and product = √2 × 5√2 = 10.

Numbers are 5 and 2. (5 + 2 = 7, 5 × 2 = 10)

Step 2: Split middle term.

√2 x² + 5x + 2x + 5√2 = 0

Step 3: Factorise by grouping.

x(√2 x + 5) + √2(√2 x + 5) = 0

(x + √2)(√2 x + 5) = 0

Step 4: Set each factor = 0.

x + √2 = 0 → x = –√2

√2 x + 5 = 0 → x = –5/√2 = –5√2/2

Answer: Roots are x = –√2 and x = –5√2/2.

Solution (iv): 2x² – x + 1/8 = 0

Step 1: Multiply throughout by 8 to remove fraction.

16x² – 8x + 1 = 0

Step 2: Find two numbers with sum = –8 and product = 16.

Numbers are –4 and –4.

Step 3: Split middle term.

16x² – 4x – 4x + 1 = 0

Step 4: Factorise.

4x(4x – 1) – 1(4x – 1) = 0

(4x – 1)(4x – 1) = 0

(4x – 1)² = 0

Step 5: 4x – 1 = 0 → x = 1/4

Answer: Root is x = 1/4 (repeated root).

Solution (v): 100x² – 20x + 1 = 0

Step 1: Find two numbers with sum = –20 and product = 100.

Numbers are –10 and –10.

Step 2: Split middle term.

100x² – 10x – 10x + 1 = 0

Step 3: Factorise.

10x(10x – 1) – 1(10x – 1) = 0

(10x – 1)(10x – 1) = 0

(10x – 1)² = 0

Step 4: 10x – 1 = 0 → x = 1/10

Answer: Root is x = 1/10 (repeated root).

Question 2

Question:

Solve the following situations mathematically:

(i) John and Jivanti together have 45 marbles. Both of them lost 5 marbles each, and the product of the number of marbles they now have is 124. Find out how many marbles they had to start with.

(ii) A cottage industry produces a certain number of toys in a day. The cost of production of each toy (in rupees) was found to be 55 minus the number of toys produced in a day. On a particular day, the total cost of production was Rs 750. Find out the number of toys produced on that day.

Solution (i): John and Jivanti marble problem

Step 1: Let John's marbles = x. Then Jivanti's marbles = 45 – x.

Step 2: After losing 5 each:

John has = x – 5, Jivanti has = 45 – x – 5 = 40 – x

Step 3: Product = 124

(x – 5)(40 – x) = 124

40x – x² – 200 + 5x = 124

–x² + 45x – 200 = 124

–x² + 45x – 324 = 0

x² – 45x + 324 = 0

Step 4: Factorise.

Find two numbers with sum = –45 and product = 324.

Numbers are –36 and –9.

x² – 36x – 9x + 324 = 0

x(x – 36) – 9(x – 36) = 0

(x – 9)(x – 36) = 0

x = 9 or x = 36

Answer:

If John had 9 marbles → Jivanti had 36 marbles.

If John had 36 marbles → Jivanti had 9 marbles.

Both are valid answers.

Solution (ii): Toy production problem

Step 1: Let number of toys produced = x.

Cost of each toy = 55 – x rupees.

Step 2: Total cost = number of toys × cost per toy

x(55 – x) = 750

55x – x² = 750

x² – 55x + 750 = 0

Step 3: Factorise.

Find two numbers with sum = –55 and product = 750.

Numbers are –30 and –25.

x² – 30x – 25x + 750 = 0

x(x – 30) – 25(x – 30) = 0

(x – 25)(x – 30) = 0

x = 25 or x = 30

Answer: Number of toys produced = 25 or 30.

Question 3

Question:

Find two numbers whose sum is 27 and product is 182.

Solution:

Step 1: Let first number = x. Then second number = 27 – x.

Step 2: Product = 182

x(27 – x) = 182

27x – x² = 182

x² – 27x + 182 = 0

Step 3: Factorise.

Find two numbers with sum = –27 and product = 182.

Numbers are –13 and –14.

x² – 13x – 14x + 182 = 0

x(x – 13) – 14(x – 13) = 0

(x – 14)(x – 13) = 0

x = 14 or x = 13

Answer: The two numbers are 13 and 14.

Question 4

Question:

Find two consecutive positive integers, sum of whose squares is 365.

Solution:

Step 1: Let first integer = x. Then second = x + 1.

Step 2: Sum of squares = 365

x² + (x + 1)² = 365

x² + x² + 2x + 1 = 365

2x² + 2x – 364 = 0

x² + x – 182 = 0

Step 3: Factorise.

Numbers with sum = 1 and product = –182 are 14 and –13.

x² + 14x – 13x – 182 = 0

x(x + 14) – 13(x + 14) = 0

(x – 13)(x + 14) = 0

x = 13 or x = –14

Since x is a positive integer, x = 13.

Answer: The two consecutive positive integers are 13 and 14.

Question 5

Question:

The altitude of a right triangle is 7 cm less than its base. If the hypotenuse is 13 cm, find the other two sides.

Solution:

Step 1: Let base = x cm. Then altitude = x – 7 cm.

Step 2: By Pythagoras theorem:

base² + altitude² = hypotenuse²

x² + (x – 7)² = 13²

x² + x² – 14x + 49 = 169

2x² – 14x + 49 – 169 = 0

2x² – 14x – 120 = 0

x² – 7x – 60 = 0

Step 3: Factorise.

Numbers with sum = –7 and product = –60 are –12 and 5.

x² – 12x + 5x – 60 = 0

x(x – 12) + 5(x – 12) = 0

(x + 5)(x – 12) = 0

x = –5 or x = 12

Since length cannot be negative, x = 12.

Altitude = 12 – 7 = 5 cm.

Answer: Base = 12 cm, Altitude = 5 cm.

Question 6

Question:

A cottage industry produces a certain number of pottery articles in a day. It was observed on a particular day that the cost of production of each article (in rupees) was 3 more than twice the number of articles produced on that day. If the total cost of production on that day was Rs 90, find the number of articles produced and the cost of each article.

Solution:

Step 1: Let number of articles = x.

Cost per article = 2x + 3 rupees.

Step 2: Total cost = 90

x(2x + 3) = 90

2x² + 3x = 90

2x² + 3x – 90 = 0

Step 3: Factorise.

Find two numbers with sum = 3 and product = 2 × (–90) = –180.

Numbers are 15 and –12.

2x² + 15x – 12x – 90 = 0

x(2x + 15) – 6(2x + 15) = 0

(x – 6)(2x + 15) = 0

x = 6 or x = –15/2

Since x must be positive, x = 6.

Cost per article = 2(6) + 3 = Rs 15.

Answer: Number of articles = 6, Cost per article = Rs 15.

Key Points – Exercise 4.2

  • Factorisation works best when roots are rational numbers.
  • Find two numbers whose sum = b and product = ac.
  • Negative lengths, ages, counts are rejected.
  • Always verify roots by substituting back.
  • If (factor)² = 0 → repeated root.

Key Formulas for Quadratic Polynomial ax² + bx + c with zeroes α and β:

  • Sum of zeroes: α + β = −b/a
  • Product of zeroes: α × β = c/a
  • To form polynomial: x² − (sum of zeroes)x + (product of zeroes)

Question 1

Question:

Find the zeroes of the following quadratic polynomials and verify the relationship between the zeroes and the coefficients:

(i) x² – 2x – 8    (ii) 4s² – 4s + 1    (iii) 6x² – 3 – 7x    (iv) 4u² + 8u    (v) t² – 15    (vi) 3x² – x – 4

Solution (i): x² – 2x – 8

Step 1: Find zeroes by factorisation.

x² – 2x – 8

= x² – 4x + 2x – 8

= x(x – 4) + 2(x – 4)

= (x + 2)(x – 4)

Step 2: Set each factor to zero.

x + 2 = 0 → x = –2

x – 4 = 0 → x = 4

So zeroes are α = –2 and β = 4.

Step 3: Verify.

Here a = 1, b = –2, c = –8

Sum of zeroes = α + β = –2 + 4 = 2 = –b/a = –(–2)/1 = 2 ✅

Product of zeroes = α × β = –2 × 4 = –8 = c/a = –8/1 = –8 ✅

Answer: Zeroes are –2 and 4.

Solution (ii): 4s² – 4s + 1

Step 1: Find zeroes by factorisation.

4s² – 4s + 1

= 4s² – 2s – 2s + 1

= 2s(2s – 1) – 1(2s – 1)

= (2s – 1)(2s – 1)

= (2s – 1)²

Step 2: Set factor to zero.

2s – 1 = 0 → s = 1/2

So zeroes are α = 1/2 and β = 1/2 (repeated zero).

Step 3: Verify.

Here a = 4, b = –4, c = 1

Sum of zeroes = 1/2 + 1/2 = 1 = –b/a = –(–4)/4 = 1 ✅

Product of zeroes = 1/2 × 1/2 = 1/4 = c/a = 1/4 ✅

Answer: Zero is 1/2 (repeated).

Solution (iii): 6x² – 3 – 7x

Step 1: Rearrange in standard form.

6x² – 7x – 3

Step 2: Find zeroes by factorisation.

6x² – 7x – 3

= 6x² – 9x + 2x – 3

= 3x(2x – 3) + 1(2x – 3)

= (3x + 1)(2x – 3)

Step 3: Set each factor to zero.

3x + 1 = 0 → x = –1/3

2x – 3 = 0 → x = 3/2

Step 4: Verify.

Here a = 6, b = –7, c = –3

Sum of zeroes = –1/3 + 3/2 = –2/6 + 9/6 = 7/6 = –b/a = 7/6 ✅

Product of zeroes = –1/3 × 3/2 = –3/6 = –1/2 = c/a = –3/6 = –1/2 ✅

Answer: Zeroes are –1/3 and 3/2.

Solution (iv): 4u² + 8u

Step 1: Find zeroes by factorisation.

4u² + 8u

= 4u(u + 2)

Step 2: Set each factor to zero.

4u = 0 → u = 0

u + 2 = 0 → u = –2

Step 3: Verify.

Here a = 4, b = 8, c = 0

Sum of zeroes = 0 + (–2) = –2 = –b/a = –8/4 = –2 ✅

Product of zeroes = 0 × (–2) = 0 = c/a = 0/4 = 0 ✅

Answer: Zeroes are 0 and –2.

Solution (v): t² – 15

Step 1: Find zeroes.

t² – 15 = 0

t² = 15

t = ±√15

Step 2: Zeroes are √15 and –√15.

Step 3: Verify.

Here a = 1, b = 0, c = –15

Sum of zeroes = √15 + (–√15) = 0 = –b/a = 0/1 = 0 ✅

Product of zeroes = √15 × (–√15) = –15 = c/a = –15/1 = –15 ✅

Answer: Zeroes are √15 and –√15.

Solution (vi): 3x² – x – 4

Step 1: Find zeroes by factorisation.

3x² – x – 4

= 3x² – 4x + 3x – 4

= x(3x – 4) + 1(3x – 4)

= (x + 1)(3x – 4)

Step 2: Set each factor to zero.

x + 1 = 0 → x = –1

3x – 4 = 0 → x = 4/3

Step 3: Verify.

Here a = 3, b = –1, c = –4

Sum of zeroes = –1 + 4/3 = –3/3 + 4/3 = 1/3 = –b/a = –(–1)/3 = 1/3 ✅

Product of zeroes = –1 × 4/3 = –4/3 = c/a = –4/3 ✅

Answer: Zeroes are –1 and 4/3.

Question 2

Question:

Find a quadratic polynomial each with the given numbers as the sum and product of its zeroes respectively:

(i) 1/4, –1    (ii) √2, 1/3    (iii) 0, √5    (iv) 1, 1    (v) –1/4, 1/4    (vi) 4, 1

Solution (i): Sum = 1/4, Product = –1

Step 1: Use formula: p(x) = x² – (sum)x + (product)

p(x) = x² – (1/4)x + (–1)

p(x) = x² – x/4 – 1

Step 2: Multiply throughout by 4 to remove fraction.

Answer: p(x) = 4x² – x – 4

Solution (ii): Sum = √2, Product = 1/3

Step 1: p(x) = x² – (√2)x + 1/3

Step 2: Multiply throughout by 3.

Answer: p(x) = 3x² – 3√2 x + 1

Solution (iii): Sum = 0, Product = √5

Step 1: p(x) = x² – (0)x + √5

Answer: p(x) = x² + √5

Solution (iv): Sum = 1, Product = 1

Step 1: p(x) = x² – (1)x + 1

Answer: p(x) = x² – x + 1

Solution (v): Sum = –1/4, Product = 1/4

Step 1: p(x) = x² – (–1/4)x + 1/4

p(x) = x² + x/4 + 1/4

Step 2: Multiply throughout by 4.

Answer: p(x) = 4x² + x + 1

Solution (vi): Sum = 4, Product = 1

Step 1: p(x) = x² – (4)x + 1

Answer: p(x) = x² – 4x + 1

Key Points – Exercise 2.2

  • Sum of zeroes = –b/a (note the negative sign)
  • Product of zeroes = c/a
  • To form polynomial: x² – (sum)x + (product)
  • Always verify answers by substituting back.
  • Multiply by LCM to remove fractions from the polynomial.
